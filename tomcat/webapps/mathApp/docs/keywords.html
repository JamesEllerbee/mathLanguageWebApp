Here you can find definitions for keywords used in this application

  • Command: The name of the command to run
  • Symbol: In the context of solve for, this means the unknown. In general, symbol is any non alphanumeric character.
  • Expression: Statement containing numbers, symbols, and operators. E.g. (x + 2) (x + 2)
  • Equation: An expression with an equality sign. E.g. 2x = 4
  • Function: An equation containing an input and output variable. E.g. y = x + 1
  • Domain: Interval notation specifying lower and upper bounds. E.g. [-10, 10]. The brackets mean these bounds are inclusive.

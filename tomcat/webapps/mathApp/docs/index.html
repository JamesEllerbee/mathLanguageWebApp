Documentation

Here you can find more about the available commands and what they do within the application.


Help

Description Displays help information for all commands or a specific command.
Usage help, help [command].

Current Mode

Description This command outputs the current output mode for the application

Change Mode

Description This command changes the current output mode for the application to the specified mode.
Usage change mode [mode] The available modes are
  • step by step
  • output
  • interactive

Simplify

Description Simplifies the given expression.
Usage simplify [expression]

Solve for

Description Performs the equation solver action.
Usage solve for [symbol], [equation]

Show graph for

Description Performs the graphing action and output an image of the graph.
Usage show graph for [functinon], [domain]
